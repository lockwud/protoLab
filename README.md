# protoLab
ProtoLab is an AI-powered Innovation Learning and Prototype Development Platform designed for the University of Skills and Entrepreneurial Development (USTED).

The platform enables lecturers to create courses, prototype assignments, and innovation challenges while helping Information Technology and Electrical/Electronics Engineering students transform project ideas into practical prototypes using AI assistance.

Unlike a traditional Learning Management System, ProtoLab focuses on innovation, project planning, prototype development, documentation, GitHub integration, and continuous lecturer supervision.
